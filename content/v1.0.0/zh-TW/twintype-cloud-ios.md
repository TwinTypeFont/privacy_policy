# TwinType Cloud iOS 隱私政策

最後更新：2026 年 5 月 8 日

本隱私政策旨在說明 TwinType Foundry LLC（以下簡稱「本公司」）在您使用 TwinType Cloud iOS 應用程式及相關服務時，如何收集、使用及保護您的資訊。我們深知隱私對您的重要性，並致力於確保您的個人資料受到妥善保護。

## 一、 解釋與定義

*   **帳戶**：指為您建立的 TwinType 使用者帳號，用於登入 App、驗證訂閱狀態及管理字體授權。
*   **應用程式 (App)**：指名為 TwinType Cloud iOS 的 iPhone 與 iPad 軟體應用程式。
*   **公司**：指 TwinType Foundry LLC（美國合法註冊公司）並與台灣字字字體有限公司合作營運，提供本地化服務支持與聯繫窗口。
*   **個人資料**：指與已識別或可識別之自然人相關的任何資訊，例如電子郵件地址、登入帳號資訊與訂閱關聯資料。
*   **使用資料**：指由服務基礎設施自動產生之技術資訊，例如裝置型號、作業系統版本、App 版本、授權驗證狀態與基本診斷資料。

## 二、 雲端服務、授權驗證與伺服器紀錄

為提供穩定且安全的字體授權服務，TwinType Cloud iOS 採用雲端帳號與授權驗證架構：

1.  **帳號登入與授權驗證**：當您登入 App 時，系統會驗證您的帳號、訂閱狀態與可用字體授權，以決定您是否可於裝置上安裝與啟用字體。
2.  **字體安裝與收回機制**：當帳號具有有效訂閱時，您可於 iPhone 或 iPad 上安裝授權範圍內之字體；當您登出、訂閱到期、取消、失敗或授權終止時，系統將停止相關授權，並使字體停用、失效或收回。
3.  **伺服器紀錄 (Log)**：當您使用服務時，伺服器可能自動記錄技術性資訊，例如 IP 位址、存取時間、裝置資訊、授權驗證結果與請求資源，用於維持系統穩定、偵測異常與處理技術問題。
4.  **雲端基礎設施**：本公司可能使用包含 Amazon Web Services (AWS) 在內之國際雲端平台、內容傳遞網路或代管服務，以執行程式、驗證授權與儲存必要資料。

## 三、 蒐集資料類型與目的

1.  **帳號與聯絡資料**：包括電子郵件地址、登入識別資訊與帳號關聯資料，用於登入驗證、訂閱管理、客戶支援與通知服務。
2.  **訂閱與授權資料**：包括目前訂閱狀態、授權可用性、字體啟用紀錄與裝置關聯資訊，用於決定字體是否可被安裝、使用或收回。
3.  **使用與診斷資料**：包括裝置型號、作業系統版本、App 版本、錯誤紀錄與基本操作資訊，用於改善穩定性、修正錯誤與提升使用體驗。
4.  **去識別化利用**：上述資訊可能在去識別化（匿名化）後，用於服務分析、效能優化與產品改進。本公司不會出售您的個人資訊，亦不會在未經同意下寄送無關之推銷內容。

## 四、 字體安裝與本機資料處理

TwinType Cloud iOS 的主要功能為讓使用者在有效授權期間內，於 iPhone 或 iPad 上安裝並使用字體。

本 App 可能於裝置本機建立與字體安裝、授權狀態或使用設定相關之必要資料，用於提供字體啟用、停用、同步與授權狀態更新功能。

當您登出帳號，或當訂閱失效、取消、到期或遭終止時，該等與授權關聯之字體可被停用、移除或收回。

## 五、 接露您的個人資料

本公司不會主動將您的個人資料提供、交換或出售給第三方，除非：

1.  **法律要求與司法機關**：在中華民國（台灣）、美國或其他適用法域之主管機關、法院或司法單位依法提出合法要求時，並於必要範圍內配合。
2.  **服務維運與安全性**：為維持帳號安全、授權驗證、防止濫用、調查異常活動、保護本公司權益或採取必要法律行動時。
3.  **委外服務提供者**：若委託第三方雲端、分析、客服、付款或基礎設施服務商協助處理必要資料，我們將盡監督之責，要求其僅於必要範圍內處理資料並遵守資安規範。
