當您完成購買、訂閱或啟用本服務，即視為已閱讀、理解並同意本條款之全部內容。

三、授權類型

（一）買斷授權（Desktop License）

1. 授權範圍

您得於桌上型電腦或筆記型電腦中安裝字型，並得將字型用於下列用途：

  1. 平面設計，如書籍、雜誌、海報、名片及其他印刷或視覺設計物。
  2. 商業印刷，且不限制印刷數量。
  3. 廣告設計，包括平面廣告及戶外廣告。
  4. 圖形輸出，包括將文字轉為外框或圖像之成果物。

2. 永久使用權

經合法購買後，您得永久使用該字型，不受後續一般性授權終止或停售之影響；但如您有違反本條款之情形，本平台仍得依本條款終止您的授權。

3. 限制事項

除本條款另有明文約定外，您不得將買斷授權字型用於下列用途：

  1. 網頁使用（Webfont）。
  2. 嵌入用途（Embedding），包括 App、軟體或遊戲。
  3. 商標註冊，不論為全部字形或部分字形。
  4. AI 訓練，包括機器學習、深度學習、字型生成、資料集建立或任何可重建字型之形式。

4. 設計服務

您得使用合法授權之字型為第三方客戶製作設計成果，但不得將字型檔本身交付、移轉、提供或授權予第三方。

（二）訂閱授權（TwinType Cloud）

1. 授權性質

訂閱授權為期間性使用權，僅於有效訂閱期間內授予使用，並不包含永久使用權。

2. 裝置限制

同一時間僅限 1 台裝置使用，且不得共用帳號、借用帳號或以其他方式讓多位使用者共同使用同一授權。

3. 使用範圍

訂閱授權之用途原則上與 Desktop 授權相同，但僅限於訂閱期間內使用。

4. 授權期間

授權期間依您所選擇之訂閱方案而定，可能為月繳或年繳。訂閱到期後，授權將自動終止，除非您另行完成續訂。

5. 到期後效果

授權到期或終止後，您不得繼續使用字型；但於授權有效期間內合法產出，且到期後未再修改之既有設計成果，得繼續保留及使用。

四、允許之產出物

於授權有效期間內，您得依授權範圍製作下列成果物：

  1. 印刷品。
  2. 廣告素材，包括影像字幕。
  3. 圖像設計，如 Logo、品牌視覺與宣傳素材。
  4. 已外框化、圖形化或無法還原為字型檔之作品。

五、需另行授權之用途

下列用途不包含於本條款所載之一般授權範圍內，您應另行向本平台取得書面或其他明確授權：

  1. Webfont。
  2. App 或軟體嵌入。
  3. 電子書銷售。
  4. SaaS 或線上設計工具。
  5. 商標註冊使用。

六、智慧財產權

除本條款明文允許外，您不得從事下列行為：

八、AI 與資料使用限制

您不得將字型用於下列任何用途：

  2. 機器學習或深度學習。
  3. 字型生成、分析模型、資料集建立或其他可推導、模擬、重建字型之技術用途。

無論商業使用或非商業使用，上述用途均一律禁止。
